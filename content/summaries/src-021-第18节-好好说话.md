# 第18节：不懂好好说话的父母，最容易养出无话可说的孩子！

## 基本信息
- 类型: 课程讲稿（音频转录）
- 作者: 沈奕斐
- 文件位置: raw/notes/沈奕斐-家庭教育课程/第18节：不懂好好说话的父母，最容易养出无话可说的孩子！.txt

## 核心论点
沟通能力是孩子未来成功的关键因素（75%靠沟通），而孩子的沟通模式由亲子沟通方式决定。父母必须从"对孩子说话"转变为"和孩子说话"——前者是命令式、寻求顺从，后者是平等式、共同思考。日常沟通有三种场景：随意聊天（用积极倾听）、传递信息（用鼓励合作五技巧）、解决冲突（先处理情绪再协商），核心原则是"don't judge"。

## 关键要点
1. **对孩子说话 vs 和孩子说话**：前者是自上而下的命令，目的是顺从；后者是平等的共同思考，目的是共识。把孩子想象成朋友就知道怎么沟通。
2. **三种沟通场景**：随意聊天（无目的）、传递信息/要求遵守规则（有目的）、解决冲突（有问题需解决）。不同场景用不同方法。
3. **Don't Judge**：聊天中不做价值判断——价值判断会切断情感流动，打消沟通欲望。聊八卦之所以开心，就是因为不需要价值判断。
4. **积极倾听四技巧**：安静专心倾听、用简单词语回应感受、让孩子说出感受、用幻想方式帮助发泄情绪。
5. **鼓励合作五技巧**：描述你所看见的/描述问题、提示、用简单语言表达原因、说出感受、改变的提醒（口头或便条）。
6. **解决冲突五步**：确认自己和孩子情绪平复 → 让孩子谈感受并表示理解 → 说出自己的感受和需求 → 一起讨论解决方案 → 如有弥补/惩罚需有相关性。
7. **惩罚必须有相关性**：把货架弄乱了就收拾（相关），不看电视一周（不相关）。惩罚是让孩子行为改变，不是增加痛苦。

## 详细笔记

### 沟通的重要性
- 卡耐基：职业人士成功75%靠沟通，25%靠天才和能力
- 孩子的沟通能力不需要上培训班，最重要的是亲子之间的沟通模式
- 父母的沟通方式潜移默化地决定孩子未来的沟通方式

### 核心认知转变：和孩子说话
- "对孩子说话"：我在思考 → 我有方案 → 我告诉你怎么做 → 目的是顺从
- "和孩子说话"：我和你一起思考 → 一起寻求解决方案 → 共同改善
- 平等 ≠ 讨好/迎合孩子：尊重孩子，也尊重自己，把自己的真实感受和想法看得很重要
- 平等 = 把孩子想象成朋友

### 场景一：随意聊天（无目的）
- **典型场景**：孩子说"今天学习特别累"
- **错误回应**：否定感受（"你有什么累的"）、急于提供方案（"你应该这样做"）、上纲上线
- **后果**：孩子不愿意聊天 → 慢慢无话可说 → 孩子在抗认成人的傲慢和自大
- **核心原则**：Don't Judge——不做价值判断，让情感自由流动
- **积极倾听四技巧**：
  1. 安静专心地倾听（专注眼神、身体语言表示愿意听）
  2. 用简单词语回应感受（"原来是这样的"）
  3. 让孩子说出感受（"这种累是怎么一种累法"）
  4. 用幻想方式帮助发泄（"如果突然出现一个外星人..."）
- **小乌龟案例**：
  - 错误：心不在焉应付 → 找原因（是不是没换水）→ 否定感受（再买一只就好了）
  - 正确：专注看着 → 回应"小乌龟死了吗" → "你感觉像失去了一个朋友？" → 幻想式安慰
- 情不通则理不达——先认同情绪，再谈其他

### 场景二：传递信息/要求合作（有目的）
- **典型场景**：要求刷牙、洗手、收拾玩具
- **错误方式**：责备、谩骂、威胁、命令、说教、警告、控诉、比较、讽刺、恐吓
- **鼓励合作五技巧**：
  1. **描述你所看见的**："我看到走道里都是玩具，影响走路了"
  2. **提示**："如果把东西整理到边上，路就出来了"
  3. **简单语言表达原因**："玩具不收拾会影响走路"
  4. **说出感受**："这让我觉得很麻烦"
  5. **改变的提醒**：口头或便条（"玩具请放回这里"）
- 目的不是操纵孩子言听计从，而是培养主动性、积极性、责任心和合作能力

### 场景三：解决冲突（有问题需解决）
- **前提**：先确保双方情绪平复——"先处理情绪再解决问题"
- **五步法**：
  1. 让孩子谈感受和需求，表示理解
  2. 说出自己的感受和需求
  3. 一起讨论，找到双方同意的解决方案
  4. 如果孩子再犯，需要做出弥补
  5. 惩罚必须有相关性
- **超市案例**：四五岁孩子跑来跑去拿东西
  - "是不是买小零食很高兴？"（理解感受）
  - "你这样我很紧张，怕你受伤"（表达感受）
  - "你可以帮妈妈推车或坐在推车上"（协商方案）
  - 信任孩子，但约定再犯的后果
- **弥补和惩罚**：
  - 弄乱了货架 → 一起整理回去
  - 工作人员批评 → 不为孩子辩护也不跟着批评，而是"妈妈陪在你身边，我们一起道歉"
  - 惩罚必须有相关性：弄乱货架→收拾（相关）；弄乱货架→不看电视（不相关）

### 不会说话的孩子
- 太小听不懂 → 直接抱走、拿走东西、坚定离开
- 用行动告诉孩子什么是合适的
- 权利与年龄吻合（[[权责统一]]）

### 沈奕斐的个人成长
- 本性急躁，有了孩子后不得不慢下来（"牵着蜗牛散步"）
- 学着理解孩子的情绪和诉求 → 脾气也变好了
- 养育孩子的过程也是自我成长的过程
