# 第14节：表扬孩子，别只知道"你真棒"！

## 基本信息
- 类型: 课程讲稿（音频转录）
- 作者: 沈奕斐
- 文件位置: raw/notes/沈奕斐-家庭教育课程/第14节：表扬孩子，别只知道"你真棒"！.txt

## 核心论点
表扬不应被否定，但需要区分真诚的表扬和虚伪的表扬。发自内心的、基于具体观察的描述性赞赏永远不会嫌多，而虚假的、带有目的性的表扬则会随孩子年龄增长被识破。表扬的本质是传递"我信任你"而非"我期待你"。

## 关键要点
1. **表扬两派之争**：一派过度表扬（虚假捧高），一派反对表扬（怕骄傲），两者都有问题。真正应该反对的是虚伪的、为了表扬而表扬的做法。
2. **真诚是前提**：中国家庭本身缺乏表扬环境，父母的真诚表扬尤其珍贵。真诚的表扬不会宠坏孩子，关键在于你是否真的看到了孩子的进步。
3. **描述性赞赏技巧**：用"描述你看见的 + 描述你的感受 + 总结为一个词"代替简单的"你真棒"。例如：不说"你真棒"，而说"我看到东西都放回原位了，我感觉很高兴，你擅长整理东西"。
4. **信任 vs 期待**：表扬传递的信息应该是"我信任你"而不是"我期待你"。期待隐含控制，信任则是基于观察的真诚肯定。
5. **RIVA表扬法**：Random（随机出现）、Intermittent（间歇性）、Varied（多样的）、Accumulated（持续的）、Reinforced（不断强化的），把表扬变成习惯。
6. **批评的原则**：批评对事不对人（"你做作业时常常停下来玩笔"而非"你怎么老是磨蹭"），表扬则要从行为落实到品质。

## 详细笔记

### 表扬的两大误区
- **过度表扬派**：看了很多育儿文章后过度表扬，孩子随手画画就捧成大画家。问题：小的时候管用，大了就没用。
- **反对表扬派**：认为表扬会导致骄傲自满，用鞭策态度对待孩子。问题：忽视了中国家庭本身就缺乏表扬环境的事实。

### 描述性赞赏的核心方法
- **三步法**：描述你看见的 → 描述你的感受 → 把行为总结为一个词
- 案例：儿子画了雨天的草地，草地画得丑但雨画得不错。不说"你简直是绘画天才"，而说"你的雨地画得很均匀，距离控制得很好，肯定是花了一点心思的"。
- 与[[优势累积法]]和[[特点论]]一脉相承：改变挑剔眼光，用欣赏的眼光看世界。

### 信任的力量
- 小王老师案例：小花被妈妈认为"没有毅力"，但老师观察发现她并非半途而废——遇到困难会暂停、想新方法、再回来尝试。老师把这个发现告诉小花，最终小花真的解决了问题。
- 信任 = 我相信你有能力（基于观察），期待 = 我希望你应该做到（隐含不信任）。

### RIVA表扬法
- **R（Random）**：表扬以一定频率出现，不是每次都要，但在关键时刻一定会出现
- **I（Intermittent）**：间歇性，隔一段时间再肯定同一件事
- **V（Varied）**：多样的，不总是表扬同一个方面，善于发现各种优点
- **A（Accumulated）**：每天持续给孩子各种肯定
- **R（Reinforced）**：同一个可复制的行为可以多次表扬，强化为孩子认识到的独特性

### 批评与表扬的对称原则
- **批评**：对事不对人，指出具体行为（可解决的），而非攻击人格
- **表扬**：从具体行为入手，落实到个人品质
- 两者都应该是真实的，但边界需要注意
