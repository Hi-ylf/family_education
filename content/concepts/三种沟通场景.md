# 三种亲子沟通场景

## 概述
亲子沟通按目的不同可分为三种场景，每种场景需要不同的沟通方法。日常生活中80%的对话是场景一（随意聊天），父母需要卸掉"教育者"功能，学会没有目的地聊天。不同场景的核心认知转变是：从"对孩子说话"到"和孩子说话"。

## 详细内容

### 核心认知转变
- **对孩子说话**：我在思考 → 我有方案 → 我告诉你怎么做 → 目的是顺从
- **和孩子说话**：我和你一起思考 → 一起寻求解决方案 → 共同改善
- 把孩子想象成朋友就知道怎么沟通
- 平等 ≠ 讨好：尊重孩子也尊重自己

### 场景一：随意聊天（无目的）
- **触发**：孩子来求助、互相没有目的只是聊天、孩子分享学校的事
- **方法**：[[积极倾听]]
- **核心原则**：Don't Judge（不做价值判断）
- **为什么Don't Judge**：价值判断会切断情感流动、打消沟通欲望。聊八卦之所以开心，就是因为不需要价值判断。

### 场景二：传递信息/要求合作（有目的）
- **触发**：要求遵守规则、不想让孩子做某事、需要孩子配合
- **方法**：[[合作五技巧]]
- **错误方式**：责备、谩骂、威胁、命令、说教、警告、控诉、比较、讽刺、恐吓
- **目的**：不是操纵孩子言听计从，而是培养主动性、积极性、责任心和合作能力

### 场景三：解决冲突（有问题需解决）
- **触发**：亲子之间有冲突需要解决
- **方法**：先处理情绪 → 积极倾听 → 鼓励合作 → 协商解决
- **前提**：确保双方情绪平复——"先处理情绪再解决问题"
- **步骤**：
  1. 让孩子谈感受和需求，表示理解
  2. 说出自己的感受和需求
  3. 一起讨论，找到双方同意的解决方案
  4. 如有弥补需要，协商弥补方式
  5. 惩罚必须有相关性

### 不会说话的孩子
- 太小听不懂 → 直接抱走、拿走东西、坚定离开
- 用行动告诉孩子什么是合适的
- 权利与年龄吻合
