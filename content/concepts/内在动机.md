# 内驱力

## 概述

内驱力是孩子学习和成长的根本动力，包含三个核心要素：兴趣、目标和成就感。学习习惯和学习方法与学习成绩的相关性背后，真正的共同因素是内驱力。父母能做的首要之事不是激发内驱力，而是不要破坏它。过大的压力会完全扼杀内驱力。

## 详细内容

### 内驱力的三个组成

来自沈奕斐课程 [src-026:3-4行]：

| 要素 | 性质 | 父母能做的 |
|------|------|------------|
| **兴趣** | 内生的 | 只能激活，不能创造 |
| **目标** | 孩子自己产生的 | 不是父母设定的；必须是孩子觉得自己能实现的 |
| **成就感** | 日常积累的 | 使用正面标签，关注具体努力和过程而非结果 |

- 目标激励的案例：初中男生为了得到心仪女孩的认同而努力学习——内驱力的出现常与对异性的兴趣同步 [src-026:3-4行]
- 父母给孩子定的目标如果孩子觉得实现不了，就失去了目标的意义

### 学习公式

智商+学习习惯+学习方法=学习成果 [src-026:2-3行]：
- 智商是居中遗传的，正态分布
- 学习习惯和方法与成绩有相关性但无因果关系
- 三者背后的共同因素是**内驱力**——有了内驱力，孩子自己能找到最适合的学习习惯和方法

### 不破坏比激发更重要

- 父母首要做的事不是激发内驱力，而是**不要破坏**内驱力 [src-026:3行]
- 太急于教好的学习方法和习惯，反倒会破坏内驱力
- 孩子小的时候，强迫形成的学习习惯本质上是外驱力，会压制内驱力的发展

### 外驱力转化为内驱力

当看不到内驱力时，先提供外驱力，通过游戏化方法转化为内驱力 [src-026:4-5行]：
- 沈奕斐案例：儿子乘法口诀逆运算不熟练→研发"考古寻宝"游戏→通过好玩有成就感的事情解决
- 家庭嗨时光：固定玩扑克牌→分解数学能力为兴趣目标和成就感
- 有内驱力时不需要强迫，孩子会主动想玩→自主性就出来了

### 压力与内驱力的关系

- 内驱力只有在**适当压力**下才能产生 [src-026:8-9行]
- 过大压力会完全扼杀内驱力
- 今天的孩子在竞争机制下特别容易压力大

**父母制造的毒性压力**（来自《自觉学成长》）：
1. 把孩子时间排得密密麻麻（以每小时/半小时为单位）
2. 忽视孩子的睡眠习惯，强调"科学睡眠时间"而忽视个体差异
3. 在孩子能力不足时依然保持高难度要求

### 天赋与内驱力

天赋与内驱力的关系 [src-025:2-3行]：
- 天赋的两个标志：强大的内驱力（哭着喊着要去学）+ 学得轻松快乐
- 如果需要强制才能学、学得很痛苦，说明天赋不够或兴趣班有问题
- 兴趣班不应该破坏内驱力——如果上了兴趣班反而失去兴趣，就违背了初衷

## 关联

- 与 [[优势累积法]] 的关系：内驱力是优势累积法的核心——关注目标而非纠错，用成就感激发动力
- 与 [[纠错教育]] 的关系：纠错教育破坏内驱力——太急于教好的学习方法反而是纠错式干预
- 与 [[归属感四阶段]] 的关系：归属感缺失→处在"寻求报复"阶段→无法产生内驱力
- 与 [[解释风格]] 的关系："没关系，努把力一定能考好"这种安慰的解释风格背后仍然是"考好很重要"的认知，无法真正解压
- 与 [[年龄段育儿]] 的关系：小学阶段题目看得懂就想帮忙→初中高中反而能放手→其实一直能给的是情感支持和能量
- 与 [[密集母职]] 的关系：把学习当成家庭生活全部内容是密集母质的典型表现，会制造过大压力
