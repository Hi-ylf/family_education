# 鼓励合作五技巧

## 概述
鼓励合作五技巧是替代责备、命令、威胁等传统方式的沟通方法，用于需要孩子遵守规则或配合行动的场景。五技巧分别是：描述问题、提示、简单表达原因、说出感受、改变的提醒。其目的是培养孩子的主动性、积极性、责任心和合作能力，而非操纵孩子言听计从。

## 详细内容

### 五个技巧

| 步骤 | 做法 | 示例（收拾玩具场景） |
|------|------|---------------------|
| 1. 描述你所看见的/描述问题 | 客观描述问题，让孩子注意力集中在问题本身 | "我看到走道里都是玩具，影响走路了" |
| 2. 提示 | 提示比谴责更容易被接受 | "如果把东西整理到边上，路就出来了" |
| 3. 简单语言表达原因 | 孩子不喜欢长篇大论，越短越记得住 | "玩具不收拾会影响走路" |
| 4. 说出感受 | 用"我觉得"而非"你怎么" | "这让我觉得很麻烦" |
| 5. 改变的提醒 | 口头或便条，文字有时比口头更清楚 | "玩具请放回这里"（便条） |

### 与传统方式的对比

| 传统方式 | 五技巧方式 |
|----------|-----------|
| "你再不收拾就不给你买新玩具了"（威胁） | 描述问题 + 提示 |
| "你是不是想让我犯心脏病"（恐吓） | 说出感受 |
| "别人家孩子每次玩完都收拾"（比较） | 简单表达原因 |
| "你不爱干净以后没人跟你做朋友"（讽刺） | 改变的提醒（便条） |

### 为什么传统方式无效
- 人本能的反应是抵触——一个不接受我的人，为什么要跟他合作？
- "我不相信你会犯心脏病""不跟他们做朋友又怎样"——这才是人的本性
- 责备、威胁、比较只会让孩子感到不被接受

### 五技巧的效果
- 孩子的负面感受减少很多
- 或许下次还会忘记，但至少不会反感提醒
- 收拾玩具的便条可能让孩子觉得收拾也是有趣的事
- 可以结合[[游戏化教育]]把收拾也变成游戏

### 如果讲了几次还不成功
- 先问自己：我的要求合理吗？孩子的年龄和能力达到了吗？需要马上完成吗？
- 回到[[优势累积法]]中的"80%的问题要走掉"
- 刷牙三分钟的例子：沈奕斐自己都做不到 → 改为一分半钟 + 流沙计时器
- 综合运用前面学到的技巧，灵活找到好的应对方式
