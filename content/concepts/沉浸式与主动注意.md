# 沉浸式与主动抓取式专注力

## 概述
专注力分为两类：沉浸式专注力（孩子天然具备，只需不破坏）和主动抓取式专注力（可有意识培养）。父母常犯的两个误区是动不动贴"缺乏注意力"标签，以及对"长时间专注"有过高期望。理解这两类专注力的区别，有助于父母正确地保护和培养孩子的注意力。

## 详细内容

### 两个常见误区
1. **动不动贴标签**：孩子上课走神、坐不住 → 贴"专注力不行"标签。但如果玩iPad很专注，就不是专注力问题，只是对要求的事没兴趣。
2. **对"长时间"期望过高**：三四岁注意力只有两三分钟是正常的；即使成人真正集中注意力也不超过10分钟。

### 沉浸式专注力
- **定义**：孩子天然具备的状态，在外界不打断、不破坏的情况下，喜欢做某事就能长时间沉浸
- **表现**：孩子很小就能盯着一个东西很长时间看
- **培养方式**：不需要额外培养，只要不破坏
- **破坏方式**：
  - 反复叫孩子名字（孩子本能回应但仍在沉浸状态，多次后不得不分精力关注你）
  - 孩子搭积木时纠正"你应该怎么搭"
  - 打断后训斥"我叫你三遍了你为什么不过来"
- **正确打断方式**：走过去拍拍肩膀 → "我现在要打断你" → 如果孩子说再等一会 → 答应并稍后再来

### 主动抓取式专注力
- **定义**：有意识地控制感觉集中在某个想要的点上
- **例子**：老师说"看这里"，有的孩子很快集中，有的游离很久
- **五个构成要素**：

| 要素 | 含义 | 培养方式 |
|------|------|----------|
| 转移性 | 五官从一个点快速转到另一个点 | 土豆快跑游戏 |
| 集中性 | 不同感觉集中在某一点并停留 | 土豆快跑游戏 |
| 延长性 | 持续时间越长越好 | 豆豆小分队游戏 |
| 广度 | 注意力打开，能看到很多东西 | 豆豆小分队游戏 |
| 分配性 | 在众多事物中有效分配注意力 | 豆豆小分队游戏 |

### 能量豆专注力套装
- **土豆快跑**：训练转移性和集中性
- **豆豆小分队**：训练广度、分配性和延长性（找只有一个不同的卡牌并连接）
- **彩虹乐园**：整合五方面能力 + 增加记忆力
- 设计特点：色彩符合孩子敏感性，五六岁孩子很快能胜过父母

### "听谁说水"游戏
- 每人右手是掌、左手是手指（放在别人掌下）
- 听到"水"字时握掌抓人 + 逃掉
- 反复训练注意力，且特别快乐
