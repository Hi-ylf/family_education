# 真实父母

## 概述

"真实父母"是沈奕斐提出的育儿理念：父母不必为了孩子完全压抑自己的真实感情，而应该给孩子一个与真实社会相吻合的"行为筐"。这个筐既不过大（过度放纵）也不过小（过度限制），既有可以说不的情况，也有坚决不能做的底线。孩子在家庭中练习的"说不"能力和行为边界，会直接迁移到社交中。

## 详细内容

### "筐"理论

父母给孩子的行为边界（筐）应该与社会的真实边界相吻合 [src-024:4-5行]：
- **筐太小**（过度限制）→孩子发现所有反抗都无用→到社会上也把自己捆绑住
- **筐太大**（过度放纵）→孩子在家想做什么就做什么→到社会上发现没人让着他→最终也是把自己捆绑起来
- **真实的筐**=有底线+有自由空间+有逻辑后果

### 问题三分法

将孩子的行为问题分为三类 [src-024:5行]：
1. **底线问题**：坚决不能做，必须约束自己（如涉及赌博、色情场所等）
2. **自然后果**：自己可以大展拳脚，自己承担结果
3. **逻辑后果**：影响他人的行为，必须约束自己

### "说不"的能力

- 孩子说不的能量，首先是跟父母在说不的时候练习起来的 [src-024:3行]
- 如果孩子经常对父母说"我不想"，父母却用各种方式让他必须服从→孩子发现反抗无用→逐渐放弃说不→到外面也不敢反抗
- 青少年案例：妈妈说很民主，每件事都给选择——但意见一致听孩子的，意见不一致听妈妈的→孩子发现选择是虚伪的→放弃选择→社交中也放弃自己的立场

### 父母展示真实感情

- 父母不必为了孩子完全压抑自己的真实感情 [src-022:16-17行]
- 可以告诉孩子"我今天心情不好，谁惹我都会生气，我需要安静一会儿"
- 父母争吵时可以告诉孩子："我们争吵跟你没有关系，也不会因此分开"
- 这种真实展示帮助孩子学习理解和应对不同的情绪状态

### 亲密关系的复制

孩子在社交中复制与父母的交往模式 [src-023:8-9行]：
- 父母习惯打击→孩子觉得"打击我才是爱我"
- 父母习惯控制→孩子觉得"被控制才是爱我"
- 父母方式"有毒"→孩子容易交到"有毒的朋友"
- 父母对孩子的情感支持模式和沟通模式会被孩子复制到未来的社交和亲密关系中

## 关联

- 与 [[权责统一]] 的关系：底线问题/自然后果/逻辑后果的三分法是权责利统一的具体应用
- 与 [[纠错教育]] 的关系：过度限制（筐太小）本质上是纠错教育——不允许孩子有自己的选择和声音
- 与 [[归属感四阶段]] 的关系：筐太小→归属感缺失→寻求报复；筐太大→不知道边界→社交中被嫌弃
- 与 [[孩子成人差异]] 的关系：六岁以下孩子分不清"拿"和"抢"，行为边界的建立需要考虑年龄特点
- 与 [[年龄段育儿]] 的关系：不同年龄段的"筐"大小应随之调整
