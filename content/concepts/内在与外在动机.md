# 内在动力与外在动力

## 概述
内趋力是由兴趣、成就感和目标所激发的动力机制，外趋力是来自外部的奖励和刺激。外趋力有其用处——可以让孩子"进入语境"——但频繁使用会导致目标偏移和不断加码。正确做法是用外趋力带入门，帮助孩子体会事情本身的好处，然后在合适时机撤掉外趋力，让内趋力接管。

## 详细内容

### 内趋力（内在动力）
- **三个来源**：兴趣、成就感、目标
- 当孩子对事情产生兴趣、在努力中体会成就感、或有自己想完成的目标时，就有满满的动力
- 内趋力驱动的行为更持久、更有创造性

### 外趋力（外在动力）
- **形式**：物质奖励（冰淇淋、钱、玩具）、积分、小红花、游戏时间等
- **正确用处**：让孩子进入原本没有兴趣的语境

### 外趋力的三个陷阱

| 陷阱 | 说明 | 案例 |
|------|------|------|
| **加码问题** | 需要不断加码才能维持同等刺激 | 一朵小红花失去吸引力 → 需要两朵 → 巧克力 |
| **目标偏移** | 从自我成长变为获取奖励 | 查理搭积木：给10块钱 → 只搭最简单的 → 创造力丧失 |
| **抵触问题** | 孩子想要的外趋常是父母不想给的 | 做好作业可以多玩游戏 |

### 查理搭积木案例（目标偏移典型）
- 爸爸说搭一个图形给10块钱
- 查理开始只搭最简单的图形赚零花钱
- 不再有创作理念和创新
- 爸爸果断中止："经济刺激让人类丧失创意的全过程"
- 正确做法：什么都不做，在孩子需要欣赏时鼓掌，具体指出哪里做得好

### 外趋力正确使用四步法
1. **进入语境**：用外趋力让孩子愿意开始做某事
2. **体会好处**：在过程中帮助孩子感受事情本身的成就感
3. **及时撤掉**：在孩子有一定内趋力后主动撤销外趋力
4. **权责利统一**：告诉孩子"你已经能承担这个责任了，可以享受这个权利"

### 积分制度实践案例
- 儿子不愿意写作业备忘录 → 设计积分制度（写一次1分，详细全做完3分）→ 10/20/30分各有奖励罐
- 发现习惯养成后主动撤掉积分 → 告诉孩子"你成长起来了" → 转为权责利统一
- 经常分享完成作业的好处（减少批评、老师满意）→ 增加成就感

### 钢琴学习中的成就感管理
- 初期兴趣：叮叮咚咚好听 → 探索欲望
- 初期成就：手指够长、学会小星星 → 激发欲望
- 困难期：弹出歌曲很难 → 需要及时成就感
- 好老师：上课学简单歌曲 + 四手联弹游戏 + 真诚肯定互补部分

### 什么时候用外趋力
- 孩子对某事没有兴趣、没有目标、没有成就感 → 用外趋力带入门
- 孩子已有内趋力 → 不需要外趋力，甚至外趋力会破坏内趋力
- 孩子有部分内趋力但需要持续 → 帮助其感受成就感而非给物质奖励

## 关联
- 与 [[游戏化教育]] 的关系：游戏化教育的目标是激发内趋力
- 与 [[权责统一]] 的关系：撤掉外趋力时用权责利统一方式过渡
- 与 [[纠错教育]] 的关系：外部控制（纠错）与外趋力类似，都会侵蚀内趋力
- 与 [[简单快乐与深度快乐]] 的关系：成就感是Hard Fun的核心来源，也是内趋力的来源
- 与 [[优势累积法]] 的关系：正面标签和具体赞赏帮助孩子感受成就感，强化内趋力
